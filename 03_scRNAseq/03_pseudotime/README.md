### Pseudotime analysis

This directory contains the pseudotime analysis used to generate **Figure 6** of the manuscript.

It includes: - An R script implementing the pseudotime analysis (`script/pseudotime.R`) - Rendered analysis reports (`pseudotime.md`, `pseudotime.html`) - Processed results used in the figure (`results/S11_table.csv`) - Session information for reproducibility (`script/sessionInfo.txt`)

All figures and tables associated with Figure 6 can be reproduced using the code and files provided in this directory.
